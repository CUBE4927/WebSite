Palette

Rule
양자 붕괴 + 순환 우위 + 셀룰러 오토마타 기반의 3단계 규칙입니다. 한 번의 시행은 붕괴(Collapse), 중첩(Superpose), 변이(Mutate) 단계로 이루어집니다.
Collapse

각 셀은 자신의 중첩된 색의 성분에 따라 R / G / B 중 하나로 확률적으로 붕괴됩니다.

  • XYZ → X / Y / Z 각각 1/3
    • RRR → R 1
    • RRG → R 2/3 / G 1/3
    • RGB → R / G / B 각각 1/3
  • 붕괴 결과는 80% 크기의 임시 셀로 표시됩니다.
Superpose

붕괴된 색 X와 변을 공유하는 이웃 셀의 붕괴된 색 Y를 비교해 다시 중첩하여 중첩된 색을 변화시킵니다.

  • 순환 우위: R > G > B > R
  • P = 0.75 × Σ compare(X,Y)
    • P > 0이면 inferior(X) |P|개가 X로 바뀝니다.
    • P < 0이면 X |P|개가 superior(X)로 바뀝니다.
    • P = 0이면 변하지 않습니다.
Mutate

각 셀의 중첩된 색의 세 성분이 각각 독립적, 확률적으로 변이됩니다. (μ' = 2/3 * μ, ν' = 1 - μ')

  • 각 성분은 확률 μ로 R / G / B 중 하나로 변이됩니다.
    • 결과적으로 성분 3개가 변이할 확률은 μ'³,
    • 성분 2개가 변이할 확률은 μ'²ν' (×3),
    • 성분 1개가 변이할 확률은 μ'ν'² (×3),
    • 모든 성분이 변이하지 않을 확률은 ν'³입니다.
