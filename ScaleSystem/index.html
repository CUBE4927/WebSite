 Z12 위의 Heptatonic Scale를 대상으로, 모든 모드에서 각 degree가 지정된 interval class를 만족하도록 정의한 회전 폐쇄형 스케일 시스템. Diatonic Scale, Melodic Major/Minor, Harmonic Major/Minor 등을 포함한, 최소한의 화성학적 기능을 보장하는 스케일 시스템이다.

Non-tertian Possibilities

 triad는 항상 해당 스케일 상의 1, 3, 5도 음을 사용하지만, 통상적인 1–3–5 구조를 가지지 않을 수 있다.

Definition

Core Idea

 하나의 7음 pitch-class set이 있을 때, 가능한 모든 tonic 선택, 즉 모든 mode에 대해 각 degree가 지정된 interval class 조건을 만족하면 그 집합을 유효한 scale family로 본다.

Implication

 이 정의는 전통적인 장/단조 기능화성에서 기대되는 1–3–5 구조를 직접 요구하지 않는다. 따라서 특정 mode에서는 sus, quartal-like, 혹은 other non-tertian tonic sonority가 자연스럽게 나타날 수 있다.
